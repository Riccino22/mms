# Movie Management System
---
This is an Odoo module for managing movies, where you can register movies, loan movies, track these loans, etc.

With this module, you can manage your movie list.
<center>
<img src="/screenshots/1.png" style="border-radius: 5px; box-shadow: 0px 0px 10px 10px grey;" width="700"></center>
<br>
You can set the movie name, description, contact who has the movie, format, genre, status, actors, and a list of contacts waiting for the movie.
<br>
<br>
<center>
<img src="/screenshots/2.png" style="border-radius: 5px; box-shadow: 0px 0px 10px 10px grey;" width="700"></center>
<br>
The application allows you to loan a movie and view the loaned movies in a list view.
<br>
<br>
<center><img src="/screenshots/3.png" style="border-radius: 5px; box-shadow: 0px 0px 10px 10px grey;" width="700"></center>

<br>
In the contacts model, you have the option to add the user to the waiting list for a movie. You also have the option to add them to the blacklist, which is a boolean field. If set to True, the user cannot borrow movies.
<br>
<br>
<center><img src="/screenshots/5.png" style="border-radius: 5px; box-shadow: 0px 0px 10px 10px grey;" width="700"></center>

<br>
You also have the option to add contacts to the waiting list for that movie.
<br>
<br>
<center><img src="/screenshots/5.png" style="border-radius: 5px; box-shadow: 0px 0px 10px 10px grey;" width="700"></center>
<br>
The module includes an actor model to keep a record of which actors appear in specific movies.
<br>
<br>
<center><img src="/screenshots/6.png" style="border-radius: 5px; box-shadow: 0px 0px 10px 10px grey;" width="700"></center>
<br>
The module also includes a model for movie formats.
